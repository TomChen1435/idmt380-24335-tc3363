# New Features Added

- Accessible Fonts
    - Use Open Sans, a sans-serif font, for all text elements
    - Avoid the use of italics at all times
- Font Size & Scaling + Line Height & Spacing + Responsive Typography + CSS clamp() Function
    - Set all headings' line-height to 150%
    - Set all paragraphs' line-height to 165%
    - Use clamp() methods when setting font sizes for mobile devices (320px <= width < 744px), tablets (744px <= width < 1024px), and desktop devices (1024px <= width)
- Line Length
    - Set max-width for all paragraph elements to 68ch
- Color Contrast
    - All text, anchor, and button elements use colors that comply with WCAG 2.1 AA Standards against the background (contrast ≥ 4.5:1)
- Color Blindness Considerations + Keyboard Navigation & Focus
    - Use a vertical offset to indicate links' hover state
- Text Alignment
    - Avoid the use of justified texts for all text elements
- Screen Reader Considerations
    - Use semantic HTML tags like "section" and "article"
    - Use descriptive texts for all links
    - Verify alt text for images
    - Include aria-label attributes when there are duplicated grouping tags
- Skip Navigation Link
    - Did not include a skip navigation link as there is no need to use one on this page
- Respect User Preferences
    - Use "prefers-reduced-motion" and "prefers-color-scheme" media queries
